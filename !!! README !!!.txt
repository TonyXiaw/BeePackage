ISB ATMA LUHUR PANGKAL PINANG 
===============================================================
[CARA INSTALASI]

1. Extract file zip Little Hope.

2. Gunakan Software yang bisa membuat "Web Server Lokal" seperti :
- Laragon atau XAMPP 
- Letakan folder "workstep" pada C/Laragon/www/"Letakan Foldernya Disini" (Jika Menggunakan Laragon)
- Letakan folder "workstep" pada C/Xampp/htdocs/"Letakan Foldernya Disini" (Jika Menggunakan XAMPP) 

3. Pastikan MYSQL aktif dan berjalan lalu buka web ketik localhost/phpMyAdmin
Setelah itu muncul tampilan Login, Masukkan :
Username : root
Password :
Setelah masuk, Buat database baru, Pilih Import, Cari dimana lokasi dari workstep.sql,
Pilih file tersebut lalu pilih import.

4. Letakan folder Mobile Android pada lokasi yang diinginkan, Jika folder tersebut zip maka extract terlebih dahulu, Lalu Buka Android Stuido pilih open dan cari dimana lokasi dari folder tersebut, Selanjutnya jika pengunduhan atau proses memuat data telah selesai, Klik : File/Settings/pilih Build, Execution, Deployment/klik Build Tools/klik Gradle/lalu pada Gradle JDK : "Pilih JDK Versi diatas 10.0.10, Pada kasus ini kami menggunakan Amazon Corretto 16.0.2. 

Tunggu proses selesai, Setelah itu jika point dari 1-4 telah dilakukan maka aplikasi tersebut sudah siap dijalankan!
